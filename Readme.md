# Toolbox for Ubuntu/Debian

## Installation

Run the following command in your terminal to install the Toolbox:

```bash
bash <(curl -s https://raw.githubusercontent.com/Julian-Zillner/Toolbox/main/install.sh)
```

## Update Toolbox

```bash
toolbox-update
```

## Update Installation Script

```bash
toolbox-install.sh

```

### Toolbox Location

/opt/toolbox
